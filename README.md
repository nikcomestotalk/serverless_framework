## Serverless API Generator

This tool helps create and deploy AWS Lambda functions with API Gateway using Serverless Framework.

### Installation

Run:

\`\`\`
npm install
\`\`\`

### Usage

#### Generate CloudFormation Template
\`\`\`
npm run generate
\`\`\`

#### Deploy to AWS
\`\`\`
npm run deploy
\`\`\`
